How to install SportsViz on iOS

It's so easy a caveman could do it.

Open up Safari on your iOS device



You need to use Safari and not Chrome (or any other browser) as we will need to install the app from Safari!

Navigate to SportsViz in your browser and stay on the home page!

If you didn't know the URL, how are you reading this?

Open up the little menu icon in the center.



See the "Add to Home Screen" button? Click that!

What that does is allows the user to add the application to their home screen, just like other apps! Since we have written code to allow the user to use this method to install SportsViz as an app!



Click the Add button in the top left hand corner...


You did it! Enjoy using SportsViz on your mobile device!

The best part? You never have to update it!
